Descripcion de los ejemplos:

Básicos:
01.IO_e_interrupciones: Parpadea un led y utiliza interrupciones para determinar el apretar de un boton.
02.Consola: Muestra lo minimo necesario para imprimir resultados en pantalla por medio de consola.
03.ManejoDeAntenaExterna: Muestra el uso de localización, donde está configurado para utilizar una antena externa.

Prueba funcional
04.PruebaFuncional: Una vez que el localizador consigue un fix, comienza a imprimir datos en un formato que permita
copiar y pegar en un visualizador.

Prueba energética
Simula los estados basicos de operacion: Bajo gasto energético esperando el apretar un botón ->
Buscando primer fix GPS, Recoger datos en intervalos regulares. El objetivo es medir los gastos energéticos en cada estado
conectando un amperimetro digital a la tarjeta durante la operación de este programa.

05.PruebaEnergetica_Base: presenta la operación sin prácticas de reducción de consumo energético.
06.PruebaEnergetica_Reducida: presenta la operación incluyendo prácticas de reducción de consumo energético.
